# MedTech Landing Page - Sovereign Clinical

This project contains the landing page for **Sovereign Clinical**, a medical technology consultancy specializing in high-tech medical equipment and hospital digital transformation in Vietnam.

## Project Overview
- **Goal**: Showcase expertise in MedTech and digital health.
- **Technologies**: HTML, Tailwind CSS, Google Fonts.
- **Key Features**:
  - Hero section with expert profile.
  - Core services: MedTech consulting, hospital digitalization, international technology transfer.
  - Portfolio of major hospital projects in Vietnam.
  - Contact form for consultation requests.

## How to Run
Simply open `index.html` in any modern web browser.
